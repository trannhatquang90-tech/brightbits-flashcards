# BrightBits Flashcards

## Quick Start (on your machine)

```bash
cd /root/workspace/apps/brightbits-flashcards
# If dependencies aren't installed:
npm install
npx expo start --tunnel
```

## Features
- Tap flashcards to reveal answers
- Kid-friendly design with emojis
- Simple MVP ready for testing

## App Structure
- `App.js` - Main component with flashcards
- `package.json` - Expo dependencies
- `dist/` - Built web version
